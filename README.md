# Neural Network Depth Analysis

## Overview
This project explores how the depth of a neural network affects its performance. 
The study compares shallow and deep neural network architectures using the MNIST dataset.

## Objectives
- Analyse how increasing network depth impacts learning
- Compare performance of shallow vs deep models
- Identify trade-offs such as overfitting and computational complexity

## Models Used
- Model A: 1 hidden layer (shallow)
- Model B: 3 hidden layers
- Model C: 5 hidden layers (deep)

## Dataset
The MNIST dataset of handwritten digits was used for training and evaluation.

## Results
- All models achieved high accuracy (~97–98%)
- Increasing depth improved learning initially
- Deeper models showed diminishing returns
- The 3-layer model provided the best balance
